# Final Report Format

At the end of setup, report in this structure.

```md
# Codex Setup Report

## Platform

- Detected platform:
- Architecture:
- Package manager:

## Existing Tools

- Tool: version

## Installed Tools

- Tool: install method

## Not Installed

- Tool: reason
- Recommended next step:

## Configuration Changes

- File or setting changed:
- Old value, if known:
- New value:

## Directories Created

- Path:

## Manual Actions Needed

- Action:
- Why it requires the user:

## Verification Summary

- Passed:
- Failed:
- Skipped:

## Notes for Future Agents

- Any machine-specific caveats:
```

Keep the report factual. Do not hide failed checks.
